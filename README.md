# NEST 就労選択支援アセスメント

就労選択支援（障害者総合支援法 第5条第13項・2025年10月開始）の支援員用アセスメント Web アプリ（単一HTML・スマホ/iPad対応・PWA）。**富士圏域共通様式に準拠**。

## 使い方
- 公開URL: https://ekazuhide-source.github.io/nest-assessment/
- iPhone / iPad の Safari で開き、共有メニューから「ホーム画面に追加」すると全画面アプリとして起動します（オフライン起動可）。
- 入力は端末内（localStorage）にのみ保存され、サーバには送信されません。複数の対象者を切り替えて記録でき、「帳票を印刷」からPDF保存できます。

## 構成
- **実施記録（富士圏域共通様式）**：「作業場面等の就労アセスメント実施記録」を再現。3領域14項目を **A/B/C/D**（各項目に評価ポイントを明記）＋コメントで評価し、**総評**（進路の見立てを含む自由記述）でまとめる。これが提出様式。
  - ① 基本的な労働習慣 ② 社会生活 ③ 職業スキル
- **支援用補足（提出様式の対象外）**：本人の希望・ニーズ／必要な配慮・環境／本人の受け止め。本人本位の選択を支える参考情報。

## 注意
- 進路（一般就労／A型／B型等）はスコアで機械的に決めるものではなく、本人と協同して選択するための参考に留めています（参考スコアは内部用）。
- 評価尺度・項目は富士圏域共通様式に準拠。様式改定時は追従します。
- 個人情報・実在の利用者データはこのリポジトリに含めません。
