# NEST 就労選択支援アセスメント

就労選択支援（障害者総合支援法 第5条第13項・2025年10月開始）の支援員用アセスメント Web アプリ（単一HTML・スマホ/iPad対応）。

## 使い方
- 公開URL: https://ekazuhide-source.github.io/nest-assessment/
- iPhone / iPad の Safari で開き、共有メニューから「ホーム画面に追加」すると、全画面のアプリとして起動します。
- 入力内容は端末内（localStorage）にのみ保存されます。サーバには送信されません。
- メニュー（⋮）から「バックアップ書き出し／読み込み」で端末間の引き継ぎ、「PDF/印刷」で帳票出力ができます。

## 構成（JEED「就労支援のためのアセスメントシート」4部構成に準拠）
- Ⅰ 就労に関する希望・ニーズ（本人本位の起点）
- Ⅱ 就労のための基本的事項（作業遂行／職業生活／対人関係。本人＝自己評価＋支援者＝協同評価）
- Ⅲ 就労継続のための環境・必要な配慮
- 協同所見・アセスメント結果票（厚生労働大臣の定める4事項の整理を含む）

## 注意
- 評価尺度（暫定4段階）と評価項目は地域裁量です。厚労省は全国一律の固定様式・尺度を定めていません。富士圏域（富士市・富士宮市）の共通様式が確認でき次第、項目・尺度を整合します。
- 進路（一般就労／A型／B型等）はスコアで機械的に決めるものではなく、本人と協同して選択するための参考に留めています。
- 個人情報・実在の利用者データはこのリポジトリに含めません。
